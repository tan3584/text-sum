## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Text-sum-web](https://github.com/tan3584/text-sum-web) repository for frontend.


## Installation

```bash
$ npm install
Or
$ yarn 
# coppy .env.example to .env and config
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
